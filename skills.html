Skills

With over 7 years of experience in computer networking, I have developed a strong foundation in managing IT infrastructure and solving complex technical challenges. Last year, I completed a program in artificial intelligence (AI), further expanding my skill set in advanced technologies.

Currently, I am pursuing Full Stack Application Development at Nova Scotia Community College (NSCC), blending my expertise in networking and AI with my passion for creating dynamic web applications.

My skills are listed below:

HTML5
HTML is the language used to create the structure of web pages, like the building blocks for websites. It tells a web browser how to display text, images, and other content.
CSS3
CSS3 is used to style and design web pages, like adding colors, fonts, and layouts. It makes websites look attractive and visually appealing.
JavaScript
JavaScript is a programming language that makes websites interactive, like adding animations or responding to button clicks. It brings web pages to life by enabling dynamic content and user actions.
Python
Python is a versatile programming language that's easy to read and write, used for tasks like automating processes and analyzing data. It helps create everything from simple scripts to complex applications like websites or AI programs.
Git
Git is a tool that helps people keep track of changes to files, like a version history for projects. It allows teams to collaborate by managing updates and merging everyone's work together.
MySQL
MySQL is a system used to store and manage data, like a digital filing cabinet for information. It helps organize, retrieve, and update data for websites, apps, or other software.
Machine Learning
Machine learning is a technology that allows computers to learn from data and improve their performance over time without being explicitly programmed. It helps create systems that can recognize patterns, make predictions, and automate tasks.
Matplotlib
Matplotlib is a library used to create visualizations and graphs from data in Python. It helps people turn complex information into easy-to-understand charts and plots.
Desktop Support
Desktop support involves helping users with their computers and software, ensuring everything runs smoothly. It includes troubleshooting issues, installing software, and providing assistance with hardware problems.
Docker
Docker is a tool that allows developers to package applications and their dependencies into containers, making it easier to build, share, and run software consistently across different environments. It simplifies the deployment process and helps manage software in isolated environments.
Virtualization
Virtualization is a technology that allows multiple virtual computers to run on a single physical machine. It enables efficient use of resources by creating isolated environments for different operating systems and applications on the same hardware.
Networking
Networking refers to the practice of connecting computers and devices together to share resources and communicate with each other. It allows information to flow between devices, enabling access to the internet, file sharing, and collaboration.
Cyber Security
Cybersecurity is the practice of protecting computers, networks, and data from unauthorized access, attacks, or damage. It involves implementing measures to secure sensitive information and ensure safe online interactions.
Bash Scripting
Bash scripting is a way to automate tasks in Unix-based systems using a command-line script. It allows users to write sequences of commands that can run automatically, making repetitive tasks easier and more efficient.
Linux Server Administration
Linux server administration involves managing and maintaining servers that run on the Linux operating system. It includes tasks like installing software, configuring networks, monitoring performance, and ensuring security to keep the servers running smoothly.
Windows Server Administration
Windows server administration involves managing and maintaining servers that run on the Windows operating system. It includes tasks like configuring settings, managing user accounts, installing updates, and ensuring security to keep the servers operating effectively.
